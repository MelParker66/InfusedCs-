# Infused C’s Website

This is the official website for **Infused C’s**, a premium infused cigar experience based in Kansas City.

## Features
- Static HTML/CSS/JavaScript site
- Event scheduling form
- Google Sheets integration for event requests
- Email notifications on form submission
- Hosted via Netlify

## Tech Stack
- HTML / CSS / JavaScript
- Google Apps Script (form handling & email)
- Google Sheets (data storage)
- Netlify (hosting)

## Deployment
This site is deployed via Netlify directly from the GitHub repository.

## Notes
No sensitive credentials are stored in this repository.
